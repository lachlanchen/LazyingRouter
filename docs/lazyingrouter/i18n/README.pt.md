# LazyingRouter

[English](../../../README.md) | [简体中文](README.zh-Hans.md) | [繁體中文](README.zh-Hant.md) | [日本語](README.ja.md) | [Français](README.fr.md) | [Deutsch](README.de.md) | [Español](README.es.md) | [한국어](README.ko.md) | [Русский](README.ru.md) | Português | [العربية](README.ar.md)

LazyingRouter é um gateway privado de API de IA para contas, chaves de API de usuários, créditos, registros de uso e roteamento para provedores de modelos.

Ele é baseado no projeto open source maduro New API. O objetivo é permitir que AgInTiFlow e outras ferramentas LazyingArt usem uma única conta LazyingRouter para acessar OpenRouter, Venice, GRSAI, serviços compatíveis com Claude e provedores futuros.

Recursos principais: login, cadastro, painel administrativo, tokens de usuário, cotas, contabilidade, logs de uso, API compatível com OpenAI, API Claude Messages, canais personalizados, roteamento ponderado e implantação Docker.

URL local: `http://127.0.0.1:3218`

URL pública planejada: `https://router.lazying.art`

Guia detalhado: [LazyingRouter operator guide](../README.md).
