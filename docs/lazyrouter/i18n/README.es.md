# LazyRouter

[English](../../../README.md) | [简体中文](README.zh-Hans.md) | [繁體中文](README.zh-Hant.md) | [日本語](README.ja.md) | [Français](README.fr.md) | [Deutsch](README.de.md) | Español | [한국어](README.ko.md) | [Русский](README.ru.md) | [Português](README.pt.md) | [العربية](README.ar.md)

LazyRouter es una pasarela privada de API de IA para cuentas, claves API de usuario, créditos, registros de uso y enrutamiento hacia proveedores de modelos.

Está basada en el proyecto open source maduro New API. El objetivo es que AgInTiFlow y otras herramientas de LazyingArt usen una sola cuenta de LazyRouter para acceder a OpenRouter, Venice, GRSAI, servicios compatibles con Claude y futuros proveedores.

Capacidades principales: inicio de sesión, registro, panel de administración, tokens de usuario, cuotas, contabilidad, registros de uso, API compatible con OpenAI, API Claude Messages, canales personalizados, enrutamiento ponderado y despliegue con Docker.

URL local: `http://127.0.0.1:3218`

URL pública prevista: `https://router.lazying.art`

Guía detallada: [LazyRouter operator guide](../README.md).
