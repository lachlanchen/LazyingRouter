# LazyRouter

[English](../../../README.md) | [简体中文](README.zh-Hans.md) | [繁體中文](README.zh-Hant.md) | [日本語](README.ja.md) | Français | [Deutsch](README.de.md) | [Español](README.es.md) | [한국어](README.ko.md) | [Русский](README.ru.md) | [Português](README.pt.md) | [العربية](README.ar.md)

LazyRouter est une passerelle privée d’API IA pour gérer les comptes, les clés API utilisateur, les crédits, les journaux d’usage et le routage vers les fournisseurs de modèles.

Elle s’appuie sur le projet open source mature New API. L’objectif est que AgInTiFlow et les autres outils LazyingArt puissent utiliser un seul compte LazyRouter pour accéder à OpenRouter, Venice, GRSAI, aux services compatibles Claude et aux futurs fournisseurs.

Fonctions principales : connexion, inscription, tableau d’administration, jetons utilisateur, quotas, comptabilité, journaux d’usage, API compatible OpenAI, API Claude Messages, canaux personnalisés, routage pondéré et déploiement Docker.

URL locale : `http://127.0.0.1:3218`

URL publique prévue : `https://router.lazying.art`

Guide détaillé : [LazyRouter operator guide](../README.md).
