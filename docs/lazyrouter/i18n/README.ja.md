# LazyRouter

[English](../../../README.md) | [简体中文](README.zh-Hans.md) | [繁體中文](README.zh-Hant.md) | 日本語 | [Français](README.fr.md) | [Deutsch](README.de.md) | [Español](README.es.md) | [한국어](README.ko.md) | [Русский](README.ru.md) | [Português](README.pt.md) | [العربية](README.ar.md)

LazyRouter は、ユーザーアカウント、ユーザー API キー、クレジット、利用ログ、上流モデルプロバイダーへのルーティングを一元管理するプライベート AI API ゲートウェイです。

成熟したオープンソース New API プロジェクトを基盤とし、AgInTiFlow と LazyingArt の各ツールが LazyRouter だけにログインすれば、OpenRouter、Venice、GRSAI、Claude 互換サービス、将来のプロバイダーを統一的に利用できることを目指します。

主な機能は、ログイン、登録、管理ダッシュボード、ユーザートークン、クレジット会計、利用ログ、OpenAI 互換 API、Claude Messages API、カスタム上流チャンネル、重み付きルーティング、Docker デプロイです。

ローカル URL：`http://127.0.0.1:3218`

公開予定 URL：`https://router.lazying.art`

詳細は [LazyRouter operator guide](../README.md) を参照してください。
