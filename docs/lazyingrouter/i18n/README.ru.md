# LazyingRouter

[English](../../../README.md) | [简体中文](README.zh-Hans.md) | [繁體中文](README.zh-Hant.md) | [日本語](README.ja.md) | [Français](README.fr.md) | [Deutsch](README.de.md) | [Español](README.es.md) | [한국어](README.ko.md) | Русский | [Português](README.pt.md) | [العربية](README.ar.md)

LazyingRouter — это частный шлюз AI API для учетных записей, пользовательских API-ключей, кредитов, журналов использования и маршрутизации к поставщикам моделей.

Он основан на зрелом open-source проекте New API. Цель — чтобы AgInTiFlow и другие инструменты LazyingArt использовали одну учетную запись LazyingRouter для доступа к OpenRouter, Venice, GRSAI, Claude-совместимым сервисам и будущим провайдерам.

Основные возможности: вход, регистрация, админ-панель, пользовательские токены, квоты, учет, журналы использования, OpenAI-совместимый API, Claude Messages API, пользовательские upstream-каналы, взвешенная маршрутизация и Docker-развертывание.

Локальный URL: `http://127.0.0.1:3218`

Планируемый публичный URL: `https://router.lazying.art`

Подробнее: [LazyingRouter operator guide](../README.md).
