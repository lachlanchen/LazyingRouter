# LazyingRouter

[English](../../../README.md) | [简体中文](README.zh-Hans.md) | [繁體中文](README.zh-Hant.md) | [日本語](README.ja.md) | [Français](README.fr.md) | [Deutsch](README.de.md) | [Español](README.es.md) | 한국어 | [Русский](README.ru.md) | [Português](README.pt.md) | [العربية](README.ar.md)

LazyingRouter는 사용자 계정, 사용자 API 키, 크레딧, 사용량 로그, 업스트림 모델 제공자 라우팅을 관리하는 프라이빗 AI API 게이트웨이입니다.

성숙한 오픈소스 New API 프로젝트를 기반으로 하며, AgInTiFlow와 LazyingArt 도구들이 하나의 LazyingRouter 계정으로 OpenRouter, Venice, GRSAI, Claude 호환 서비스 및 향후 제공자를 사용할 수 있게 하는 것이 목표입니다.

주요 기능: 로그인, 가입, 관리자 대시보드, 사용자 토큰, 쿼터, 과금/회계, 사용량 로그, OpenAI 호환 API, Claude Messages API, 커스텀 업스트림 채널, 가중 라우팅, Docker 배포.

로컬 URL: `http://127.0.0.1:3218`

예정 공개 URL: `https://router.lazying.art`

자세한 내용: [LazyingRouter operator guide](../README.md).
